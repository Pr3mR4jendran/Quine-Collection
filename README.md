# Quine-Collection
A quine is a program that takes no input and prints its own source code. This repository is a collection of quines in various programming and scripting languages.

Key for file extensions:
.cpp - C++
.cs - C#
.html - HTML
.java - Java
.js - Javascript
.m - Matlab
.pl - Prolog
.py - Python
.r - R
.sh - Bash
